# Python Problem Solving

A structured collection of Python solutions focused on developing strong problem-solving skills, algorithmic thinking, and coding intuition.

This repository is built through consistent practice of programming problems, with emphasis on:

* Pattern Recognition
* Logical Thinking
* Mathematical Reasoning
* Time & Space Complexity Analysis
* Code Optimization
* Clean and Readable Implementations

## Key Concepts Practiced

* Digit Extraction (`%10`, `//10`)
* Factor Pair Optimization (`√n`)
* Prime Number Techniques
* Euclidean Algorithm (GCD/LCM)
* Two-Pointer Technique (reversal, palindrome checks)
* Hashing / Frequency Maps
* Recursion Fundamentals
* Pattern Recognition
* Complexity Analysis

## Learning Approach

Each problem is solved with the following mindset:

1. Understand the underlying pattern.
2. Build intuition before coding.
3. Write a working solution.
4. Analyze Time Complexity.
5. Analyze Space Complexity.
6. Optimize wherever possible.
7. Maintain clean and readable code.

The objective is not only to solve problems but also to understand *why* a solution works and *when* a particular pattern should be applied.
