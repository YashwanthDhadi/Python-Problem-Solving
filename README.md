## Learning Approach

Each problem is solved with the following mindset:

1. Understand the underlying pattern.
2. Build intuition before coding.
3. Write a working solution.
4. Analyze Time Complexity.
5. Analyze Space Complexity.
6. Optimize wherever possible.
7. Maintain clean and readable code.

The objective is not only to solve problems but also to understand *why* a solution works and *when* a particular pattern should be applied.
